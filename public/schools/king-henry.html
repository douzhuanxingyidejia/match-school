学校简介

亨利八世国际学校位于雪兰莪赛城，是一所英式寄宿与走读制国际学校，面向3至18岁学生。学校设有幼儿、低年级、中学与预科阶段，并提供 IGCSE 与 A-Level 升学路径。

英国姊妹校背景

学校与英国 Christ College Brecon 具有紧密联系，延续英式学校结构、学院文化与升学取向。

📘

国际考试授权路径

学校提供 Cambridge International 与 Pearson Edexcel 相关资格课程，为 IGCSE 与 A-Level 评估做准备。

课程体系

3–5岁

EYFS

面向 Nursery 与 Reception 阶段，围绕英国早期教育框架的七大学习与发展领域开展启蒙学习。

5–11岁

Key Stage 1–2

小学阶段重视英语、数学、科学与专题学习，并通过专科课程支持阅读、表达与基础学科能力。

11–14岁

Key Stage 3

中学低年级拓展核心学科与专科课程，为后续 IGCSE 科目选择和学术方向建立基础。

14–16岁

IGCSE

Key Stage 4 阶段以 IGCSE 课程与考试为核心，学生在核心科目外结合兴趣选择考试科目。

16–18岁

A-Level

预科阶段提供 AS 与 A-Level 学习路径，学生根据大学专业方向选择科目并接受升学指导。
