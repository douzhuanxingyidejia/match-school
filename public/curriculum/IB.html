体系简介

IB 课程体系是一套覆盖小学、中学到大学预科阶段的国际课程路线，通常包括 PYP、MYP 与 Diploma Programme。该体系强调探究式学习、跨学科理解、全球视野、研究写作和综合能力发展，帮助学生从低龄阶段逐步过渡到高强度的大学预科课程与全球大学申请。

3–12岁

PYP

小学项目以探究式学习为核心，帮助学生建立好奇心、表达能力、基础学科理解和跨主题学习习惯。

11–16岁

MYP

中学项目强调学科广度、概念理解、跨学科连接和项目实践，为后续 DP 阶段建立综合能力基础。

15–16岁

Personal Project

MYP 高年级通常会通过个人项目探究学生的研究、计划、执行和反思能力，衔接更高阶学习要求。

体系特点

🎓

探究式学习贯穿全程

IB 从 PYP 到 MYP 都强调探究式学习，学生通过问题、主题和项目理解知识之间的联系。

📊

强调跨学科与全球视野

IB 不只关注单一学科成绩，也重视学生对不同文化、社会议题和全球问题的理解与表达。

🧩

DP 阶段保持学科广度

IB DP 要求学生覆盖语言、人文、科学、数学和艺术等领域，同时选择 HL 与 SL 科目形成深浅搭配。

🌎

核心项目要求突出

TOK、EE 与 CAS 是 IB DP 的重要组成部分，能够体现学生的研究能力、反思能力和综合发展。

PYP 到 DP 阶段重点

PYP 与 MYP

从探究能力到学科广度
  • PYP 以探究单元为核心，帮助学生建立表达、合作和自主学习习惯。
  • MYP 扩展语言、人文、科学、数学、艺术和设计等学科广度。
  • 学生通过跨学科任务和项目实践理解知识在真实情境中的应用。
  • 这个阶段重点不是过早定专业，而是形成学习方法和综合素养。

IB DP

通过六大学科组与核心项目衔接大学申请
  • DP 阶段通常学习六大学科组，并区分 Higher Level 与 Standard Level。
  • 学生需要完成 TOK、Extended Essay 和 CAS 三项核心要求。
  • 课程要求学生兼顾考试成绩、研究写作、时间管理和活动参与。
  • DP 成绩与课程组合通常会成为全球大学申请的重要参考。

评分与学习方式

IB ASSESSMENT

综合评估学科表现

IB 体系在不同阶段结合课堂任务、项目、内部评估和外部考试。DP 阶段尤其重视学生在各学科中的理解、分析、表达和应用能力。

INQUIRY-BASED LEARNING

强调跨学科与全球视野

PYP 与 MYP 阶段重视探究、项目、展示和反思，学生需要在真实情境中提出问题、收集信息并表达自己的理解。

DP CORE REQUIREMENTS

DP 核心要求影响表现

IB DP 不只看单科成绩，也要求学生完成 TOK、EE 与 CAS。研究写作、批判思考和时间管理能力会显著影响整体表现。

INQUIRY-BASED ROUTE

适合主动探究型学生

如果学生愿意主动提问、阅读、讨论、做项目和表达观点，IB 的探究式学习方式更容易发挥优势。

GLOBAL ROUTE

目标全球大学方向

如果学生希望未来保持英国、美国、加拿大、澳洲、香港、新加坡等多方向申请弹性，IB DP 的全球认可度较高。

BALANCED ROUTE

综合能力较均衡

如果学生语言、写作、数学、科学、人文和活动参与能力较均衡，IB 的广度要求和核心项目更容易形成综合竞争力。

规划建议

PYP 阶段

重点建立阅读、表达、合作、好奇心和探究习惯，不宜过早只关注考试结果。

MYP 阶段

通过多学科学习、项目任务和跨学科活动观察学生优势，逐步建立研究与反思能力。

DP 选课前

提前判断 HL 与 SL 科目组合，匹配未来专业方向、大学要求和学生真实能力。

IB DP 阶段

同步管理六门学科、TOK、EE 与 CAS，保持时间管理、写作和考试表现稳定。

相对其他体系的特点

更强调综合素养

IB 相比只聚焦少数学科或单次考试的体系，更强调学生的学科广度、研究能力、表达能力、反思能力和社会参与。

全球认可度较高

IB DP 被全球许多大学认可，适合希望保持多国家、多地区申请弹性的学生。

学习强度与管理要求高

IB DP 要求学生同时兼顾六大学科组和核心项目，对时间管理、持续写作和抗压能力要求较高。

适合全面发展型学生

如果学生不只追求单科成绩，也愿意参与研究、服务、活动、表达和跨学科学习，IB 更容易体现整体发展。
